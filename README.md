# Daily-Rewards

A configurable plugin that gives items to players once a day with a command

# Items

```yaml
---
Items:
- "306:0:1"
- "307:0:1"
- "308:0:1"
- "309:0:1"
...
```

This is the default items config, You can change the items/armor reward by getting their ID, Meta and count
